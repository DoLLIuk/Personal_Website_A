# Portfolio Project

This is a modern portfolio starter built with Vite, React, TypeScript, Tailwind CSS and shadcn-ui components.

## Quick start (local)

Ensure Node.js and npm are installed, then run:

```bash
npm install
npm run dev
```

Open http://localhost:8080 in your browser.

## Technologies

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui

## Building for production

```bash
npm run build
npm run preview
```

Feel free to edit the source files in `src/` to customize the site.
